# Quiz Backend

## Scripts

```shell
# Run unit tests
pnpm run test

# Run linting, code formatting and typechecks
pnpm run lint
pnpm run format
pnpm run typecheck

# Run the local dev Netlify server
pnpm run dev

# Prisma scripts
pnpm run prisma:migrate
pnpm run prisma:generate
pnpm run prisma:studio
```
